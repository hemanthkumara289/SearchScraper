# 🔍 SearchScraper

A Python-based web scraping application that automates search queries using Selenium WebDriver and extracts search result information. This project demonstrates browser automation, dynamic web interaction, and basic web scraping techniques.

---

## 🚀 Features

- Perform automated search queries
- Extract search result titles and links
- Browser automation using Selenium
- Automatic WebDriver management
- Handles dynamic page loading using explicit waits
- Easy to extend for other websites

---

## 🛠️ Tech Stack

- Python 3
- Selenium
- WebDriver Manager
- Microsoft Edge WebDriver

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/SearchScraper.git
cd SearchScraper
```

### 2. Create a virtual environment (Optional)

Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

Linux/macOS

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python main.py
```
